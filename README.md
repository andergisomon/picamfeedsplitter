# Pi Camera Feed Splitter

A vibecoded camera feed splitter for the Raspberry Pi utilizing ffmpeg, mediamtx,
and iceoryx2 for single-copy duplication of N streams. vibecoded with Claude Code.

## Application
Developed for [Palanuk](https://github.com/Lagadan-ISDP-G5/palanuk)

## Intended usage

The main binary is to be run before the binary in examples/webrtc_streamer.rs is run.
